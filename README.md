# Отмазки (мобильная веб-игра)

Статическая игра на чистом HTML/CSS/JS, оптимизированная под мобильный экран.

## Локальный запуск

Открой `index.html` в браузере
или запусти локальный сервер:

```bash
cd /Users/karenyaylyan/Desktop/mila
python3 -m http.server 8000
```

Открой: `http://localhost:8000`

## Публикация на GitHub Pages

1. Залей файлы этого проекта в репозиторий (в корень).
2. В GitHub: `Settings` -> `Pages`.
3. В `Build and deployment` выбери:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` (или `master`), папка `/ (root)`
4. Сохрани и подожди 1-2 минуты.
5. Открой ссылку вида:
   - `https://<username>.github.io/<repo-name>/`
